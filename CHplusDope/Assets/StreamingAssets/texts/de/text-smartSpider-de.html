Der smartspider

von smartvote


"Der smartspider ist eine Visualisierung von acht thematischen Ratings. Die acht Ratings orientieren sich an zentralen Themen der Schweizer Politik.
Die Berechnung der Positionen auf diesen Achsen erfolgt anhand der Antworten beim smartvote-Fragebogen. Dazu werden die Fragen in der Regel einer oder mehreren Achsen zugeordnet und ein Rating berechnet. Einzelne Fragen sind keiner Achse zugeteilt.
Pro Achse kann ein Wert zwischen 0 und 100 erreicht werden. Ein Wert von 100 steht für eine starke Zustimmung zum formulierten Ziel der Achse. Ein Wert von 0 bedeutet, dass das formulierte Ziel keine Zustimmung erhält."
